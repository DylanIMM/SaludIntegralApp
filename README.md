# Sistema Distribuido de Gestión de Clínicas

Sistema web desarrollado para la administración de una red de clínicas utilizando **Bases de Datos Distribuidas** sobre **SQL Server**

## Descripción

El sistema permite administrar la información de dos sedes de una clínica:

- **Salud Integral Norte**
- **Salud Integral Centro**
---

# Tecnologías utilizadas

## Backend

- Node.js
- Express.js
- SQL Server
- mssql (Node SQL Server Driver)

## Frontend

- HTML
- CSS
- JavaScript

## Base de Datos

- Microsoft SQL Server
- SQL Server Management Studio (SSMS)

---

# Funcionalidades

- Gestión de Clínicas
- Gestión de Médicos
- Gestión de Pacientes
- Gestión de Consultorios
- Gestión de Citas

# Características Destacadas

- Arquitectura distribuida
- Linked Servers.
- Vistas Particionadas.
- Stored Procedures.
- Transacciones Distribuidas mediante MSDTC.

---
Proyecto académico desarrollado para la asignatura de **Bases de Datos Distribuidas**.
